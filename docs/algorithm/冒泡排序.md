# 冒泡排序


![网上找的图](/640.png "网上找的图")

```bash
function sort(arr, xs) {
  let flag = true  // 是否还需要继续判断标志
  for (let i = 1; i < arr.length; i++) {
    flag = true
    for (let j = 0; j < arr.length - 1; j++) {
      if (arr[j] > arr[j+1]) {
        [arr[j], arr[j+1]] = [arr[j+1], arr[j]]
        flag = false
      }
    }

    if (flag) {
      break
    }
  }

  return arr
}
```
